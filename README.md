Event Dashboard
A lightweight event management dashboard built with React.
This project focuses on organizing events, tracking details, and presenting them in a clean, responsive UI suitable for admin-style workflows.

Overview
The dashboard allows users to view and manage event data in a centralized interface. It was built as a front-end focused project with emphasis on component structure, state management, and usability.

Key goals of the project:
Build a scalable React component structure
Practice dashboard-style layouts
Handle dynamic data rendering
Create a responsive UI for event management

Tech Stack
React
JavaScript (ES6+)
HTML5 / CSS3
